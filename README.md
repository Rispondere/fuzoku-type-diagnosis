# 風俗嬢タイプ診断システム

## 概要
キャストとスタッフの個性を分析し、効果的な店舗運営をサポートする診断システムです。

## 機能
- キャスト向けタイプ診断（4タイプ分類）
- スタッフ向け指導タイプ診断
- 相性分析と指導マニュアル

## 使用方法
1. [メインページ](https://yourusername.github.io/fuzoku-type-diagnosis/)にアクセス
2. 診断したい対象を選択
3. 質問に回答して結果を確認

## ファイル構成
- `index.html` - メインページ
- `cast-diagnosis.html` - キャスト向け診断
- `staff-diagnosis.html` - スタッフ向け診断
- `staff-manual.html` - 指導マニュアル
